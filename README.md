# QueueSmart

## Setup & run

### 1. Clone the repo

```bash
git clone <repo-url>
cd QueueSmart
```

### 2. Install dependencies 

From repo root:
```bash
pnpm run install:frontend
```

Or from the frontend folder:
```bash
cd frontend
pnpm install
```
### 3. Run dev server

From repo root or frontend:
```bash
pnpm dev
```
